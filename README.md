# Exam Portal Application

A full-stack exam platform built with React, Node.js, and Supabase.


## Technologies
- **Frontend**: React (Vite), TypeScript
- **Backend**: Node.js/Express
- **Database**: Supabase (PostgreSQL)
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
